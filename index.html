Setup Instructions

Getting up and running takes less than 30 seconds.

1

Copy the Command

Select your operating system tab above, then click the Copy button to copy the curl/powershell installation command to your clipboard.

2

Run in Terminal

Open your command line interface (Terminal on Linux/macOS, PowerShell on Windows) and paste the command. Hit Enter to execute it.

3

Start Organizing

The installer pulls the codebase, installs packages, compiles scripts, and runs the app on your system, launching the web interface automatically.
